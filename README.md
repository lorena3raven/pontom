
# Ponto M Moda Íntima

Landing Page desenvolvida para a **Ponto M Moda Íntima**, com foco em conversão, experiência premium e atendimento via WhatsApp.

## ✨ Sobre o projeto

A proposta da landing page é apresentar a marca de forma elegante e sofisticada, destacando seus principais produtos e incentivando o contato imediato através do WhatsApp.

O projeto possui design responsivo, animações suaves e identidade visual voltada para o segmento de moda íntima e bem-estar.

## 🚀 Tecnologias utilizadas

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons

## 📱 Funcionalidades

- Hero Section responsiva
- Botão flutuante do WhatsApp
- Categorias de produtos
- Diferenciais da empresa
- Como funciona
- Produtos em destaque
- Chamada final para conversão (CTA)
- Footer com redes sociais
- Design totalmente responsivo
- Animações suaves
- SEO básico

## 📂 Estrutura do projeto

```
app/
components/
lib/

```

## ▶️ Executando o projeto

Instale as dependências:

```bash
npm install
```

Execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abra no navegador:

```
http://localhost:3000
```

## 📦 Build

```bash
npm run build
```
